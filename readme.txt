Сборка Webpack, собраннная с помощью старших товарищей для дальнейшего использования в проектах.
В качестве препроцессора CSS используется SASS.
Поддерживается полноценный PostCSS конфиг с автопрефиксером, упаковщиком медиазапросов и минификацией CSS-кода. Имеется пока что закомментированная возможность написания HTML кода в виде PUG-файлов с дальнейшей компиляцией.
На стороне JS подключен Babel 7-й версии, с автоматической транспиляцией кода в формат ES5.
Подключена среда разработки Vue.js.
